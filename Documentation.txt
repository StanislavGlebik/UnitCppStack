Документация по стеку
-------------------------------------

1 Компиляция проекта
    В связи с тем что проект написан на с++, компиляция проекта осуществляется с помощью команды "make".
	Приложение использует библиотеку cppunit для юнит тестирования.
	Посмотреть как собирается проект можно в makefile.
2 Очистка проекта
    Очистка проекта осуществляется с помощью того же makefile, командой make clean.
3 Функциональность проекта
    Проект реализовывает stl стек.
4 Структура проекта
	Основной класс - MyStach.
	Заголовочный файл - stack.h.
	Функции:
		конструктор по умолчанию,
		конструктор копирования,
		Size,
		Pop,
		Top,
		Push,
		деструктор.
	Что делают функции можно увидеть из названия, параметры такие же как в stl стеке, только для целочисленных параметров.
